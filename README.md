# 🚀 Northwind SQL Business Analysis

## 📌 Project Overview

This project showcases my SQL skills by analysing the Northwind database to solve real-world business problems. Using MySQL, I explored customer behaviour, product performance, supplier activity, sales trends and operational insights through a series of business-focused SQL queries.

Each query includes the SQL solution, query output, and a business interpretation to demonstrate not only technical SQL knowledge but also the ability to extract meaningful insights from data.

---

## 🎯 Project Objectives

This project answers key business questions such as:

- Which products generate the highest profit?
- Which customers place the most orders?
- Which products have never been ordered?
- Which suppliers provide the most products?
- Which products generate the highest revenue?
- Which countries contribute the most customers?
- What are the sales and pricing trends across products?
